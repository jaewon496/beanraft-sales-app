[정확히 이대로 만들 것]

맨 위 제목: "카페 위치 지도"
우측 상단: 반경 숫자 표시(예: 500m) + 슬라이더 + X 닫기 버튼

필터 바:
- 좌측: "345개" 한글 카운트 (파란색 강조)
- 우측: 정확히 5개 필터 칩 (한 줄에 가로 배치)
  칩1: 빨간 점 + "검색 위치"
  칩2: 파란 점 + "프랜차이즈"
  칩3: 초록 점 + "개인카페"
  칩4: 주황 점 + "베이커리"
  칩5: 보라 점 + "신규 오픈"

지도 영역:
- 지도 plat 위에
- 중심에 빨간 머그컵 아이콘 + "강남역 1번 출구" 라벨
- 중심 주변에 하늘색 원형 반경 표시 (500m 경계)
- 원 안에 작은 머그컵 마커들 (필터 색상별)
- 우측 상단에 지도 줌 +/- 버튼만

[마커 팝업 열림 프레임만 추가 요소]
마커를 탭하면 흰 말풍선이 뜸. 내용:
- 카페명 (예: "스타벅스 강남R점")
- 주소 (예: "서울 강남구 역삼동 814")
- "중심에서 120m"

[빈 상태 프레임]
지도 중앙에 ⓘ 아이콘 + "반경 내 카페가 없습니다" + "반경을 늘려보세요"
재검색 버튼이나 다른 버튼 넣지 마.

[에러 상태 프레임]
경고 아이콘 + "지도를 불러올 수 없습니다" + "네이버 지도 인증에 실패했습니다"
"다시 시도" 버튼 (투명 배경 + 흰 테두리)

[로딩 프레임]
